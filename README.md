# CPLBank ATM System

This is a simple console-based ATM banking simulation developed in C++.  
The program allows users to log in using a card number and PIN to perform basic banking operations such as withdrawals, deposits, transfers, and balance checks.

## Features
- Secure login using card number and PIN
- Withdraw and deposit money
- Check account balance
- Transfer funds between accounts
- Change account PIN
- Mobile top-up and bill payment
- Fast cash withdrawal
- Transaction history and receipt generation

## Technologies
- C++
- File handling (Accounts, transaction history, receipts)
- Standard C++ libraries (iostream, fstream, vector, etc.)

## Purpose
This project was created as part of a programming assignment to practice file handling, structured programming, and building a menu-driven banking system.
